## 🚀 下一步操作

### 1. 安装 Node.js
```bash
# 访问 https://nodejs.org/ 下载 LTS 版本
# macOS 用户可以使用 Homebrew:
brew install node

# 或下载安装包直接安装
```

### 2. 启动项目
```bash
# 方法一：使用启动脚本（推荐）
./start.sh

# 方法二：手动启动
npm install
npm start
```

### 3. 开始使用
1. 启动应用后，点击"导入文件"或"导入文件夹"
2. 选择您的音乐文件开始播放
3. 探索各种功能：创建播放列表、收藏歌曲、切换主题等
